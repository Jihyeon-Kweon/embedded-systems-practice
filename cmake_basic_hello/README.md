# cmake_basic_hello

This is a simple "Hello, World!" project to practice the full build flow using CMake and Makefiles.

# Project Structure
cmake_basic_hello/
├── CMakeLists.txt
├── src/
│ └── main.c
├── build/ # build directory created by CMake
└── README.md

# Build Instructions
```
mkdir build && build
cmake ..
make
./hello
```
